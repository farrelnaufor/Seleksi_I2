Fitur Pengujian:
  - Pencarian Valid: Menginput kata kunci pekerjaan dan memastikan hasil muncul.
  - Pencarian Tidak Valid (Negative Test): Menginput kata kunci acak dan memastikan hasil pencarian kosong.
  - Filter: Memilih opsi tertentu dari dropdown dan memverifikasi pemilihan filter.
  - Navigasi Menu News: Membuka halaman berita dan memverifikasi konten.
  - Pembukaan Sosial Media: Mengakses tautan Instagram di footer dan memastikan halaman berhasil terbuka.
  - Screenshot Otomatis: Setiap pengujian menghasilkan screenshot sebagai dokumentasi.
